# Glossary

Common terms used throughout this guide.

| Term | Definition |
|---|---|
| **AppId** | Application ID from Imou Open Platform console |
| **AppSecret** | Secret key used to sign API requests. Keep private. |
| **bindDeviceLive** | API call to create a cloud-proxied live stream |
| **Channel** | A video input source. For NVRs, each physical port is a channel. |
| **Device ID** | Unique identifier for a camera/NVR in the Imou system |
| **DPS** | Device Password Seed — Imou's password derivation system |
| **HLS** | HTTP Live Streaming — a playlist-based streaming format compatible with browsers |
| **NVR** | Network Video Recorder — records and manages multiple IP cameras |
| **ONVIF** | Open Network Video Interface Forum — standard protocol for IP cameras |
| **P2P** | Peer-to-Peer — Imou's NAT traversal for mobile app connectivity |
| **RTSP** | Real Time Streaming Protocol — standard for video stream delivery |
| **Safety Code** | Default password printed on the camera's physical label |
| **Subtype** | Stream quality selector: 0=main/HD, 1=sub/SD |
| **WS-Discovery** | Web Services Dynamic Discovery — used by ONVIF to find devices on LAN |
